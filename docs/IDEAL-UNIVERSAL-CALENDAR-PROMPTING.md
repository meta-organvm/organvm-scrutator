## THE 7-DAY PROMPT CYCLE

### Day 1: DISCOVERY
**Prompt:** "What exists? List everything without judgment."
- No filtering
- No structure
- Pure witness
- **Output:** Raw inventory

### Day 2: ORGANIZATION
**Prompt:** "Group what exists. Find patterns."
- Surface atoms only
- Basic categorization
- **Output:** Organized atoms

### Day 3: DECOMPOSITION
**Prompt:** "Break each atom into subatomic elements."
- ENTITY / ACTION / DOMAIN / REFERENCE / TIMESTAMP / PRIORITY
- **Output:** Decomposed elements

### Day 4: MULTIVERSAL EXPLORATION
**Prompt:** "If TRUE, FALSE, UNKNOWN, ALWAYS, NEVER—what follows?"
- Explore logic instances
- Map implications forward and backward
- **Output:** Multiversal map

### Day 5: SYNTHESIS
**Prompt:** "What is the PURE IDEA? Extract the universal principle."
- Generalize beyond context
- Find what applies everywhere
- **Output:** Principles

### Day 6: IMPLEMENTATION
**Prompt:** "Build the work queue. Make it actionable."
- Tasks with priorities
- Blockers identified
- Dependencies mapped
- **Output:** IRF-ready items

### Day 7: REFLECTION
**Prompt:** "What did the system learn? What improved?"
- Compare to previous week
- Measure pathos/logos/ethos shifts
- **Output:** Vital signs update

## THE THREE FILTERS APPLIED TO PROMPTING

### FILTER 1: RAW (Intention)

Before typing anything:

1. **Capture the impulse exactly** — Don't polish, don't structure
2. **Use my voice** — Direct, unfiltered, conversational
3. **Preserve urgency** — The emotional charge matters
4. **Examples:**
   - "What did we do so far?"
   - "Get crazy if you want"
   - "These are like stats that will live, like blood pressure"

**Exercise:** Write 3 versions of your raw prompt. Keep the most honest one.

---

### FILTER 2: FORMALIZED (Context)

Transform raw into domain-specific:

1. **Add context** — What repo? What session? What's the state?
2. **Structure for action** — Commands, questions, statements separated
3. **Tag for tracking** — Priority, domain, dependencies
4. **Example transformation:**

   **RAW:** "check the IRF for P0"
   
   **FORMALIZED:**
   ```
   Context: Session S-2026-04-26, organvm-corpvs-testamentvm
   Task: Query IRF for P0 items
   Action: organvm irf list --priority P0
   Output format: markdown table
   ```

**Exercise:** Take your raw prompt and add: context line, action verb, format specification.

---

### FILTER 3: PURE IDEA (Universal)

Extract what's eternal:

1. **Ask "Why does this matter beyond today?"**
2. **Find the principle that applies everywhere**
3. **Generalize until it hurts**
4. **Examples:**

   | Raw Prompt | Pure Idea |
   |------------|-----------|
   | "What did we do?" | "Continuity requires artifact-level memory" |
   | "Get crazy" | "The system must measure its own ignorance" |
   | "Plan everything" | "Every unknown is a research opportunity" |

**Exercise:** After every prompt, ask: "What's the universal principle here?"

## THE PROMPT QUALITY SCORE

Rate each prompt 1-10 on:

| Dimension | Question | Weight |
|-----------|----------|--------|
| **Clarity** | Could someone else understand this? | 2x |
| **Context** | Is the domain specified? | 1x |
| **Actionability** | Can an AI act on this directly? | 2x |
| **Depth** | Does it survive Filter 3? | 2x |
| **Voice** | Is this authentically my voice? | 1x |

**Formula:** Score = (Clarity×2 + Context×1 + Actionability×2 + Depth×2 + Voice×1) / 8

**Benchmarks:**
- 8-10: Masterful prompt
- 6-7: Strong prompt
- 4-5: Average prompt
- 1-3: Needs work

## COMMON PROMPT FAILURES

### Failure 1: Skipping Filter 1
**Bad:** "Run the scan"
**Good:** "I need to see what's changed since yesterday — check all .claude/plans/ directories and report what we missed"

### Failure 2: Skipping Filter 2
**Bad:** "What's the system health?"
**Good:** "Using organvm-scrutator, calculate aggregate vital signs (pathos/logos/ethos) for the 8 tracked entities. Report scores and trend arrows."

### Failure 3: Skipping Filter 3
**Bad:** "Update the IRF"
**Good:** "Add IRF-SYS-161 with the pure idea: 'Governance vital signs should be tracked like personal health metrics — continuous, comparable, actionable.'"
